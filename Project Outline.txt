Title: Technical Term Translator

- GUI presents the option to choose language, then presents a text field for input and a space for output
- API handles the calls to the medical dictionary and returns the "translated" text
- translation handles translating text from English to other languages (i.e. German, Spanish, etc.)
- textToSpeech takes text and returns the audio file of it spoken
