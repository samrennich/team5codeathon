Python
 1 go to https://www.python.org/downloads/release/python-3104/
 2 download the appropriate installer
 3 follow the instructions in the wizard
 4 run "py -V" to verify python has been successfully installed and your version is up-to-date

Git
 1 go to https://git-scm.com/book/en/v2/Getting-Started-Installing-Git
 2 follows all instructions for appropriate OS
 3 verify in terminal with "git --version"
 4 in any directory, type "git clone https://github.com/SamRennich/team5codeathon.git"
 5 before every time to go to edit/add code, type "git pull" when in the repository to receive all new updates
 6 type "git add *file names go here*", "git commit -m "message goes here"", and "git push" when making adding/editing code
